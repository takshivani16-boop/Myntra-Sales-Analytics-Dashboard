# 🛍️ Myntra Sales Analytics Dashboard Overview

An interactive Power BI dashboard 
built to analyze Myntra sales data 
and generate business insights across 
categories, brands, regions, discounts, 
and customer behavior.

## ✨ Features
- ✅ Sales KPI Overview
- ✅ Brand-wise Orders Analysis
- ✅ Monthly Sales Trends
- ✅ State-wise Sales Distribution
- ✅ Category & Size Insights
- ✅ Discount & Ratings Analysis
- ✅ AI-powered Key Influencers Visual
- ✅ Interactive Slicers and Filters

## 🛠️ Tools & Technologies
- Power BI
- Power Query
- DAX
- Data Modeling

## 📁 Dataset
- Time Period: 2021-2023
- Total Orders: 3,500+
- Unique Customers: 100
- Categories: Women, Men, Kids, Beauty
- Tables: fact_orders, dim_customers,
  dim_products, Date_table

## 💡 Business Insights
- Identified top-performing brands 
  and categories
- Analyzed sales trends and 
  customer preferences
- Compared category contribution 
  and regional performance
- Explored factors influencing 
  total sales

## 🚀 How to Use
1. Download .pbix file
2. Open in Power BI Desktop
3. Use slicers to filter data
4. Navigate between Overview 
   & Data Analysis pages
